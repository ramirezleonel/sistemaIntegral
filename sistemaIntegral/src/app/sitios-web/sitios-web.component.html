Sitios Responsive

Es una técnica de diseño web que busca la correcta visualización de una misma página en distintas pantallas. Es decir, éste término hace referencia al diseño web multidispositivo. En la actualidad, los usuarios acceden a los sitios web desde diferentes dispositivos, ya sean ordenadores de sobremesa, portátiles, iPads, tablets o smartphones y smart TV.


Además existen elementos de navegación que se deberán mostrar diferente,como por ejemplo el menu de navegación, resultando más cómodo para el usuario y por ende, más intuitivo.
